# WebSpace

 
![demopic]( our demo pic here ) 


### Description
WebSpace it's a network facilitator application that helps you to get an overview of all attendees in a event.



**Key words**: 

*React.js,*
*MongoDB,*
*Amazon Cognito,*
*AWS,*
*Amplify,*
